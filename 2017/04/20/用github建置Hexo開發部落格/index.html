Github 與 SSH 連結 (前置)

建立一個專案

專案名稱為 username.github.io

GitConfig

產生SSH

此時應該會跳出訊息

複製 rsa金鑰 github

在github的帳號設定(Personal settings)裡面有個 SSH and GPG keys
在SSH keys 點選 New SSH Keys

點選add SSH key

確認SSH key 是否正常

第一次會出現選項 請打yes

出現下面訊息代表成功，此時就可使用ssh 存取github
